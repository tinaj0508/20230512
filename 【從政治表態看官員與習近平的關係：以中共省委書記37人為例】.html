【從政治表態看官員與習近平的關係：以中共省委書記37人為例】

附錄：文字探勘探索性分析

本論文附錄旨在針對表態指標、文本主題進行探索性的分析，希望利用word2vec、tf-idf與stm三種演算模型，探索下面兩個目標:

  1. 改善正文中的表態指標關鍵詞
  2. 了解官員講話中的主題分布，計算提及習近平主題的比例。透過機器分類出的習近平主題文本，幫助筆者以質性內容分析方法閱讀文本時，縮小閱讀範圍；幫助筆者分析不同主題文本的用詞。

  1. 省委書記表態篇數(2021年至2022年按月分排序)
  2. 省委書記表態篇數(按人名依照講話篇數排序)

二、Word2Vec

下面以"习近平",“两个确立”,“核心”,“灯塔”,"掌舵"五個表態指標中的關鍵詞舉例，使用word2vec將詞彙轉換為向量，獲得相似度高的詞彙，希望可以找到更多相似的表態關鍵詞。

model_w2v <- word2vec(Data$segcon)
nn <- predict(model_w2v,
c("习近平","两个确立","核心","灯塔","掌舵"),
type="nearest",top_n = 30)
View(nn$掌舵)

1. 掌舵

從掌舵來看，相似度高的詞如「領航、指路、眾望所歸、定向、舉旗、把舵」等等，這些都是用來形容習近平指引方向的詞彙。

2. 習近平

習近平的相似詞出現的結果可以讓筆者思考「黨的十八大精神、黨的十九大精神、習近平新時代中國特色社會主義思想」這類意識形態的詞彙，是否要納入表態指標中。另外，有些斷詞如「領會好」、「深學」需要被改善。

3. 兩個確立

做為口號類的習近平變形詞，可以看到出現的相似詞都與黨中央權威、歷史地位、習近平有關。

4. 核心

核心的相似詞彙中出現幾個令人意外的詞彙，「光谷」、「海絲」、「元器件」等等，回到文本看，有些核心指的是在產業中的核心地位，並非指習近平。

5. 燈塔

回到文本看，只有蔡奇與李鴻忠會以「燈塔」形容習近平。由於中共官員講話會用的形容詞都蠻相似的，或許可以將「燈塔」的相似詞納入關鍵詞，看看其他官員會不會用類似的詞彙形容習近平。

三、tf-idf

1. 省委書記+習近平文本所有詞彙不分詞性之tf-idf

涉及的主題很多，出現的關鍵詞都是大方向，看起來對表態指標沒有幫助。因此筆者在第二點中會根據官員與詞性抽取tf-idf分析。

2. 單一省委書記短文本詞性之tf-idf

單一官員文本中，抽取提及「“总书记”,“习近平总书记”,“两个确立”,“两个维护”,“核心”,“灯塔”,“掌舵”,“领袖”」之句子成為短文本，並以每個短文本為單位，從動詞與成語抽取tf-idf前10名。最後將所有短文本tf-idf前10名的動詞與成語加在一起計算頻率。希望可以從提及習近平的短文本中，抽取動詞與成語，發現省委書記用什麼動詞去擁護習近平，或者，如果其中的動詞與擁護無關，可以顯示省委書記在提及習近平時的講話風格為何。

import jieba
import jieba.analyse

data1 = data[data["Name"]=="李鴻忠" ]
text = []
for i in range(0,len(data1["content"])):
    for e in range(0, len(data1["content"].iloc[i].split("。"))):
        text.append(data1["content"].iloc[i].split("。")[e])
Xitext=[]
matches = ["总书记","习近平总书记","两个确立",
"两个维护","核心","灯塔","掌舵","领袖"]
for i in range(0, len(text)):
    if any([x in text[i] for x in matches]):
        Xitext.append(text[i])
        print(text[i])

verblist = []
for i in range(0,len(Xitext)):
    keyword = jieba.analyse.extract_tags(
    Xitext[i],topK=20,
    withWeight=True,allowPOS=('i','v'))
    for r in range(0, len(keyword)):
        verblist.append(keyword[r][0])

import numpy as np

e = pd.value_counts(np.array(verblist))

(2) 以許勤講話為例:

從圖片中可以看到，在短文本中被列入tf-idf高頻詞最多的是「深入、振興、貫徹、堅持」等動詞與成語，在許勤的文本中，雖然與李鴻忠講話中出現的詞彙十分相似，但出現「振興」、「打造」兩個詞，與擁護習近平較無關的詞彙。
筆者在閱讀文本時也發現，許勤講話中雖然也會擁護習近平，但比較像是形式上的擁護，講話內容更多的還是談及省內發展建設問題。這一觀察與tf-idf呈現的關鍵詞能相互映證。

舉例來說，許勤提到習近平的同時，主要還是想講振興黑龍江。

要认真学习贯彻习近平总书记关于加强和改进统一战线工作的重要思想及多党合作的重要论述，坚守合作初心，勇担历史使命，各民主党派省委领导班子要团结带领广大民主党派成员矢志不渝跟党走、携手奋进新时代，不断开创民主党派工作新局面，为推动龙江全面振兴全方位振兴作出新贡献

但是李鴻忠講話中提到習近平的句子中，即使同樣在講省內發展，但會用比較多的篇幅去講習近平

我们深刻领悟“两个确立”的决定性意义，增强“四个意识”，坚定“四个自信”，做到“两个维护”，不断增强政治判断力、政治领悟力、政治执行力，坚定不移推进习近平新时代中国特色社会主义思想在津沽大地的扎实实践，天津发展已经站在一个新的历史起点上

四、STM

藉由stm建立topic model，希望可以得到兩個方向:
(1) 省委書記與習近平在某些主題中的遠近度。
(2) 主題建模中的關鍵詞能提供表態指標關鍵詞的參考。
(3) 省委書記講話中，提及習近平主題與其他主題的關係。

(一)、以省委書記+習近平文本下去跑stm

首先，以篇為單位，選擇Held out likelihood相對較低， 語意連貫性(Semantic Coherence)較高、 殘差(Residuals)相對平穩的區間，最後使用25個topic。

findK <- searchK(documents = out$documents, vocab = out$vocab,
prevalence = ~ NameN, 
K = c(10,15,20,25,30,35,40,45,50,55,60,65,70), 
init.type = "Spectral", data = out$meta, core = 1)
